# Assets Folder

This folder should contain app icons and splash screens.

## Required Files (Generate these):

1. **icon.png** - App icon (1024x1024 px)
2. **adaptive-icon.png** - Android adaptive icon (1024x1024 px)
3. **splash.png** - Splash screen (1242x2436 px)
4. **favicon.png** - Web favicon (48x48 px)

## How to Create Icons:

### Option 1: Use Online Generator
- Visit: https://www.appicon.co/ or https://easyappicon.com/
- Upload a logo or design
- Download the generated assets
- Replace the files in this folder

### Option 2: Use Design Software
- Design your icon in Figma, Photoshop, or Canva
- Export at the required sizes
- Place in this folder

### Option 3: Use Expo's Default Icons (Temporary)
- The app will use Expo's default icons if these files don't exist
- This is fine for development and testing

## Recommended Icon Design:

For GoMate, create an icon featuring:
- Map pin or location marker
- Sri Lankan theme (maybe a palm tree or landmark)
- Blue color scheme (#2563EB)
- Simple, recognizable design

## Splash Screen Design:

- Background: Gradient from #2563EB to #3B82F6
- Center: "GoMate" logo/text
- Subtitle: "Explore Sri Lanka"

---

**Note:** The app will work without custom assets - Expo provides defaults. Add custom icons before final submission for a more professional look!
